PONG GAME

Godot Engine - pong game

 

Godot Engine is a free, open-source game development tool that helps people create 2D and 3D games. Its designed to be user-friendly, allowing you to build and design games without needing a lot of experience. We used Godot to make a simple pong game, where two paddles bounce a ball back and forth

From this, I learned how games are made and how coding can bring them to life. It was really cool to see my own game work after building it step by step. Using Godot helped me understand how even simple games require logic and creativity and it made me excited about game development
