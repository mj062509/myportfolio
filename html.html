HTML

HTML (HyperText Markup Language) is the basic language used to create websites. It helps structure the content on a webpage, like organising text, images and links. When we used HTML, we created lists, paragraphs, headings and other elements to make a webpage easy to read and navigate.

From this, I learned how simple it is to create lists using HTML. I now understand how websites are built step by step and it feels great to know I can organise content on a page. Its exciting to see how these small pieces of code work together to create something people use every day.
